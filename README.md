Ionic App 基础 框架
=====================
> 本框架包含ionic的基础框架内容

> 基于gulp + browersync可实现js css的自动注入和修改文件后自动刷新

> 以及多终端同步测试

## 安装配置
=====================
### 预先安装
> 预先安装node.js 
> 安装全局gulp
```bash
$ sudo npm install -g gulp
```

### 程序安装
```bash
$ npm install
```

## 运行方式
```bash
$ gulp
```


浏览器会自动打开http://localhost:3000端口
